Welcome to ACACIA

Acaca is a micro webservice for managing CaltechAUTHORS submissions via DOI.

Workflows

  • EMail
    1. Get Messages
    2. Messages to DOI
    3. Review message list
  • Data Entry
    1. Add a DOI
  • Processing submitted DOI
    1. Retrieve Metadata
    2. Review DOI list
    3. For each DOI record in the list
      1. Review the DOI, Object URL and any existing EPrints record
      2. If no-EPrints record exists click the Export link “EPrint XML” and save locally
      3. Download any related PDF for digital materials from either the DOI link or Object URL link (if one is provided)
      4. Import the EPrint XML and PDF into CaltechAUTHORS
      5. Confirm EPrints import was successful
      6. In the DOI row in Acacia click the “remove” button to remove the row from the list
