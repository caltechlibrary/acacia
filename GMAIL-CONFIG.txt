
If you're using GMail as the host for the email address
there is some additional setup you need to do.

1. Turn on IMAP this is done from the GMail settings when logged into th eacocunt.
2. You need to enable "Allow Less secure app access", this is done from
the going to "https://myaccount.google.com/u/1/security" clicking
the "security" menu item on the left, scrool down to "Less Secure app access" in the main content area, the click on the "Less secure app access" title and enable it (move the dot to the right so it is blue).
